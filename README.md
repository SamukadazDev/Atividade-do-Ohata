# Projeto Calculadora - Qualidade de Software

Atividade prática desenvolvida para a disciplina de Qualidade de Software, com foco em refatoração de código, tratamento de exceções, documentação técnica e versionamento com Git/GitHub.

## 🛠️ Tecnologias Utilizadas
- Java
- IDE Eclipse
- Javadoc (Geração de documentação em HTML)
- Git e GitHub

## 🚀 O que foi feito no projeto?
1. **Refatoração:** Substituição de estruturas condicionais extensas por uma estrutura limpa utilizando `switch-case`.
2. **Tratamento de Erros e Exceções:** Implementação de validações para capturar divisões por zero e operadores inválidos usando `IllegalArgumentException`.
3. **Testes Unitários Manuais:** Criação de uma classe de teste (`TesteCalculadora.java`) para validar os cenários normais e os cenários de exceção no console.
4. **Documentação Técnica:** Geração completa das páginas de documentação da API utilizando a ferramenta Javadoc (arquivos salvos na pasta `/doc`).

## 📋 Como Executar os Testes
A classe `TesteCalculadora.java` contém o método `main`. Ao executá-la como uma Java Application no Eclipse, o console exibirá os resultados das operações e o tratamento das mensagens de erro.
